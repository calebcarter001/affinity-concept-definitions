# Boutique - Theme Definition
## Overview
- **UUID**: f81df649-263c-4692-a94f-035a97228296
- **Owned by**: Manuela Sammatrice
- **Last Updated**: Feb 13, 2025
- **Automation**: No
- **Attribute ID**: 5022
- **URN**: BoutiqueTheme

## Definition
This theme is to be assigned **manually** to properties that feature **unique and upscale decor**.

Guestrooms are individually furnished and feature added touches, often with locally influenced décor and artwork.  
Boutique properties typically have fewer than 100 rooms and are mostly unbranded. Some may belong to designated hotel collections (e.g., Design Hotels, Preferred Hotels).

### Special Rule for Greece
Due to official classification, **hotels in Greece** only qualify if official classification is applicable.  
If a property in Greece qualifies for the Boutique Theme, it should be assigned both the following URNs:
- `BoutiqueTheme`
- `HellenicChamberBoutiqueHotel` (Boutique Award)

## Exceptionally Included
On request, in preparation for possible Theme automation:
- **Attribute ID**: 1073745236
- **URN**: BoutiqueTheme:AuthorizedManualAddition

## Not Included
- **Turkey**: Properties in Turkey are not eligible for this theme due to official classification policies. See Turkey policy documentation for details on Boutique and Special Class actions.
- **Greece**: If a property **does not** provide an official Boutique certificate but wants to use the word “Boutique” in its name, assign the URN `HellenicChamberBoutiqueHotel:Unavailable`.

If a partner requests exclusion from the theme filter, select the `THEME OVER-RIDE, RMV BOUTIQUE` attribute.  
(Note: This is precautionary since the theme is not yet automated.)
